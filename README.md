# miniguia-projeto-growth
O tema abordado no projeto "Metodologia Growth: Framework para Escalar Negócios" partiu da necessidade real de comprender como funciona a metodologia e como é possivel a aplicabilidade favorecendo o impulsionamento nos negócios. Para estruturar o projeto através do NotebookLm, foram utilizados fontes de pesquisas da web, tais como: repositórios, livros, artigos e outras materias disponibilizadas no google, posteriormente abrangendo mais os recursos de pesquisas através da deep search fornecido pelo próprio notebooklm.
As fontes primarias para referencia de pesquisa foram
https://v4company.com/blog/estrategias-de-marketing/o-que-e-growth-marketing
https://br.hubspot.com/blog/marketing/growth-marketing-sem-hacks
https://bib.pucminas.br/pergamumweb/download/A3C3D68E-01AB-444E-8703-43470007EA5D.pdf
https://troylendman.com/2025-growth-loop-case-studies-breakthrough-strategies-revealed/
Os principais insights que tive usando o NotebookLM foram sobre como ele ajuda a organizar melhor as ideias e deixar o conteúdo mais claro, durante seu uso, ficou mais fácil entender a metodologia de Growth, conectar teoria com prática e perceber pontos como oportunidades de melhoria e falhas no que estava sendo analisado. Outro ponto que fez diferença foi a agilidade, pois a propria ferramenta ajudou bastante a resumir conteúdos mais complexos que tornou o processo de estudo mais fluido e menos confuso. Evidentemente o resultado depende muito de como a gente usa a ferramenta, embora o uso de prompt facilite os meios de pesquisa, é de suma importancia revisar, interpretar e filtrar as informações para garantir que os insights realmente fizessem sentido no contexto do projeto, no demais, o NotebookLM é um ótimo apoio para organizar e acelerar o aprendizado, mas claro, sem invalidar o olhar crítico por trás da aplicabilidade.
